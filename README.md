## 需求
1. 白名单限制
2. 接口能满足本次调用【剩余次数、有效期内】
3. QPS限制

## 中间件
1. rabbitmq
2. redis
3. mysql

## 技术
1. springboot
2. mybatis
3. email
4. 定时任务（单击）
5. 集群分布式任务

## 功能
1. 接口白名单（允许用户限制某个接口的白名单,限制白名单内的ip通过，可配置）
2. 接口限流（允许用户限制某个接口的qps，可配置）
3. 接口使用限制（限制接口剩余次数和生效失效时间段通过，可配置）
4. 接口安全（每个用户针对每个接口都有一个appkey）
5. 接口计费模式
    - 按次收费
    - 试用
    - 免费
    - 包年
    - 包月
    - 自定义时间
6. 使用次数为0或者过了失效时间的用户可以自定义是否需要邮件、短信通知。
